什么时候需要一个项目？通常我们在标准的“离线”项目下工作。这个项目隐含包括了全部页面和文件夹。在这个项目里，你能够“直接发布”新建或修改过的页面和文件夹。你还能发布整个离线项目。但是对于大型网站，你也可能会发布了超出你权限（或意识）之外的资源。

子项目可以把网站按照多个作者来分解，或作为不同的各自独立的部分。比如你的网站提供稳定的内容，你经常做些小的修改，如修改错别字、添加一些新闻。在子项目中，你能添加全新的、需要一些时间才能固定的内容。这样的话，频繁发布“离线”项目也不会影响正在开发的新内容。

 

如果发布“离线”项目，这不会导致其它项目的内容被发布，即使其它项目的内容包含在“离线”项目的根文件夹下。

 

在这个页面里，你能创建新项目、修改当前项目的设置（在作业区左上角选择）或浏览当前项目的历史记录。
